# Text-Summarizer
# Installation
### To clean the Data library
- Numpy 1.22.1
`pip install numpy`
- String 0.1.2
`pip install strings`
- Re 2021.11.10
`pip install regex`

### To save the model library
- Pickle 5
`pip install pickle5`

### Neural Network library
- TensorFlow 2
Requires the latest pip
`pip install --upgrade pip`

Current stable release for CPU and GPU
`pip install tensorflow`
- Keras 2.7.0
`pip install keras`

### Natural Language Processing library
- NLTK 3.6.7
`pip install nltk`
- SpaCy 3.2.1
`pip install spacy`
